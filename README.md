# Apollo Movies

Movie app built with React, Apollo and GraphQL

## Set Up
1. npx create-react-app 프로젝트명
2. cd 프로젝트명
3. code . 으로 프로젝트 오픈
4. terminal에서 필요한 패키지 설치 (yarn add styled-components react-router-dom apollo-boost @apollo/react-hooks graphql)
5. npm start
